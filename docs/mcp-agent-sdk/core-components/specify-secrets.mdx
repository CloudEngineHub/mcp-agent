## Overview

mcp-agent provides several ways to manage secrets securely:

1. **mcp_agent.secrets.yaml** - Gitignored file for local development
2. **Environment variables** - For cloud deployments
3. **MCP_APP_SETTINGS_PRELOAD** - For production without plaintext files

## Using Secrets Files

```yaml mcp_agent.secrets.yaml
openai:
  api_key: "sk-..."

anthropic:
  api_key: "sk-ant-..."

custom_service:
  username: "user"
  password: "pass"
```

<Warning>
  Always add `mcp_agent.secrets.yaml` to your `.gitignore` to prevent committing secrets.
</Warning>

## Environment Variables

```bash
export OPENAI_API_KEY="sk-..."
export ANTHROPIC_API_KEY="sk-ant-..."
```

Reference in config:

```yaml mcp_agent.config.yaml
openai:
  api_key: ${OPENAI_API_KEY}
```

## Production Secrets

For production deployments, use `MCP_APP_SETTINGS_PRELOAD`:

```bash
export MCP_APP_SETTINGS_PRELOAD='{"openai": {"api_key": "sk-..."}}'
```
