## Quick Start

Deploy to mcp-agent cloud in 3 commands:

```bash
# Install CLI
uv tool install mcp-agent

# Log in
mcp-agent login

# Deploy
mcp-agent deploy my-agent
```

[See Deployment Quickstart →](/deployment/deployment-quickstart)

## Use Cases

<Tabs>
  <Tab title="Deploy Agents">
    Deploy complete agent applications with workflows and MCP servers:

    ```bash
    mcp-agent deploy my-agent-app
    ```

    Your agent becomes accessible as an MCP server at:
    ```
    https://deployments.mcp-agent.com/servers/my-agent-app
    ```
  </Tab>

  <Tab title="Deploy MCP Servers">
    Deploy standalone MCP servers to the cloud:

    ```bash
    mcp-agent cloud servers deploy my-mcp-server
    ```

    [Learn more →](/deployment/mcp-agent-cloud/deploy-mcp-server)
  </Tab>

  <Tab title="Deploy ChatGPT Apps">
    Expose agents as ChatGPT custom actions:

    [See OpenAI Integration →](/deployment/use-cases/deploy-chatgpt-apps)
  </Tab>
</Tabs>
