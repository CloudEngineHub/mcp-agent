## Cloud Architecture

mcp-agent cloud provides managed infrastructure for deploying agents:

- **MCPApp as Server** - Your agents are exposed as MCP servers
- **Temporal Backend** - Durable execution for long-running workflows
- **Container Deployment** - stdio MCP servers run as containers
