## Using Deployed Servers

Configure access to your deployed agent:

```bash
# Configure the deployed server
mcp-agent cloud configure --id https://deployments.mcp-agent.com/servers/my-agent

# Or use mcp-agent install command
mcp-agent install my-agent
```
